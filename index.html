Orthofuzz -    
A java library for functional annotation and pathway analysis of denovo assembled RNA-Seq data
http://orthofuzz.github.io/
Nikolau Group - BBMB/BCB - Iowa State University

  • Overview:

    The Orthofuzz package positions contigs assembled de-novo from RNA-Seq short read data among protein sequences obtained from phylogenetic neighbors of the target organism in a muti-organism pairwise sequence similarity network using pairwise BLAST scores.

    Keywords:

    source organisms: organisms, the annotations of which are used as sources of functional annotations.

    source sequences: sequences from the source organisms.

    target organism: the denovo sequenced organism, which we are analyzing.

    target sequences: sequences from target organisms.

    List of External Packages used:

    A series of java programs which uses network topology and edge weights are then utilized to carry out analysis of the RNA-Seq data. The package include the following interdependent java programs:

    A set of organisms in the phylogenetic neighborhood of with assigned functional annotations are chosen and All vs All BLAST carried out on translated and protein sequences to define an All vs All pairwise sequence similarity network.

    The topological neighborhood of the pairwise sequence similarity networks and BLAST bit scores are then used to transfer functional annotations from source organisms to targets, summarize gene-expression values of gene sets and predict interaction relationships between the denovo contigs.

    Key concepts [Manuscript under preparation]:

    1. The fragmented nature and the high identity between sub-groups of the contigs make the current approach of assignment of contigs to the best hits in the source organisms [Table X] error-prone [Figure X].

    2. Protein sequences from the same gene family share more neighbors with higher edge weights compared to those that just share functional domains [Figure X].

    3. Random walking with restart on an edge-weighted network captures high edge weights and topological neighborhood to give probabilistic definition of network neighborhood [Figure X].

    There are four algorithms at the core of the package [Refer Manuscript]:

    1. Random walk with restart on an edge weighted network starting from a set of start nodes.

    2. Estimate summary expression value of the topological neighbors of a set of nodes.

    3. Transfer functional annotations from source organisms to target organisms.

    4. Transfer interactions from source organisms to the target organisms.

    5. Modified Tomita Algorithm to identify representative cliques in sequence similarity networks.

    1. TransferAnnotations.jar: Transfers functional annotations from the source sequences to the target sequences.

    2. SummarizeExpressions.jar: Summarizes expression levels of contigs highly homologous to a set of query sequences.

    3. Uses the SummarizeExpression.jar to estimate expression levels of enzymes in metabolic pathways.

    4. GeneSetEnrichmentStats.jar: Generate Gene set enrichment statistics of enzymes and metabolic pathways.

    5. PredictInteractions.jar: Generates hypothesis about interaction networks and protein complexes, which are known to interact with each other and is supported by co-expression of their members.

    6. GeneListStatistics.jar: Generate Gene list statistics of lists of contigs to identify functional annotation terms enriched in lists of contigs.

    7. ParalellCoexpression.jar: Estimates pearson correlation of expression data using paralellColt java package.

    8. FindRepresentativeCliques.jar: Finds representative cliques in a graph.

    1. Preparing the input files

    2. Run Network Flow program to estimate the strength of homology between the source and target sequences.

    3. Run the SumExpression program to estimate the representative expression levels for query gene sets from the complied pathways.

    4. Run the TransferAnnotations program to transfer the annotations from the source sequences to the target sequences.

    5. Run the TransferInteractions program to estimate evidence for the presence and strength of protein interactions of the source organisms in the target organisms.

    6. Run Coexpression program on the expression of the interaction data to identify interaction units that are co-expressed.

    10. Run NetworkAND program to identify protein complexes that are interacting and are co-expressed in the target organisms.

    11. Run Coexpression program on the pathway expressions to identify enzymes and pathways that are co-expressed.
